# Karabiner Mods

Custom Karabiner-Elements configurations for AI-assisted development on macOS. Caps Lock becomes a dual-purpose key (tap for Escape, hold+key for 40 text shortcuts) covering workflow prompts, thinking shortcuts, slash commands, research delegation, and session knowledge persistence. All shortcuts are designed for composability — atomic single-intent prompts that chain together for complex operations like end-of-session capture (persist → synthesize → audit).

Beyond keyboard shortcuts, the project includes Xbox Wireless Controller mappings (25 actions via standalone buttons and LB/RB combos), mouse button shortcuts (Razer Viper 8KHz with app-specific overrides), and SuperWhisper voice dictation integration. Uses Svorak keyboard layout — JSON configs map to QWERTY physical key codes.
